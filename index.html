rem-point

Soft-grid classes using rem.

What

186 selectors for layout on an .5-rem-based soft-grid.

Example

Here's an example of the media object with all measurements removed and replaced with `rem-point` classes.

Chantastic

Some jabbering on about the most pendantic nonsense you've ever heard.

Open this in your inspector.

Stats

186 selectors, 1069B gzipped, all your layout needs.

Implementation

  • .5rem as the smallest unit of measurement.
  • Incremental scale: 1x, 2x, 3x, 4x, 5x, 6x, 7x, 8x
  • minions.css style class naming.
